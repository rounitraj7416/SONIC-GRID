# SONICGRID AI — Real-Time Urban Acoustic Intelligence

SONICGRID transforms cities from passive listeners into adaptive acoustic ecosystems that predict, prevent, and heal urban noise pollution in real time.

## Features

- **Edge AI Collection**: Smartphones, IoT sensors, and smart poles extract acoustic features on-device. Raw audio is never uploaded — only anonymized metrics.
- **Privacy Processing**: Federated learning, differential privacy, and acoustic fingerprint hashing ensure zero speech reconstruction. Geo-fuzzing protects locations.
- **AI Prediction Engine**: Graph Neural Networks fuse acoustic data with traffic, weather, and event calendars to predict noise hotspots 30–120 minutes ahead.
- **Automated Response**: Traffic rerouting, quiet zone activation, and citizen alerts deploy automatically.
- **Quiet Home Planner**: Use acoustic intelligence to discover the best noise-free neighborhoods for your dream home.

## Tech Stack

- **Frontend Framework**: Vite
- **Styling**: Vanilla CSS
- **Libraries**: Chart.js, GSAP, Leaflet, Three.js

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```

## Architecture

- **On-Device ML & TinyML** for privacy-first Edge AI.
- **Temporal ML & GNN** for real-time predictions.
- **Multi-Modal Integration** linking soundscapes with AQI, crowd density, heat, and sleep data.
